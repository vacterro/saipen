# 貢献者行動規範 (Contributor Covenant Code of Conduct)

## 私たちの誓約

私たちメンバー、貢献者、そしてリーダーは、年齢、体格、目に見える、あるいは目に見えない障害、民族、性徴、性同一性および性表現、経験レベル、教育、社会経済的地位、国籍、容姿、人種、宗教、または性的指向および性自認にかかわらず、私たちのコミュニティへの参加をすべての人にとってハラスメントのない体験にすることを誓約します。

私たちは、オープンで、歓迎され、多様で、包摂的で、健全なコミュニティに貢献する形で行動し、交流することを誓約します。

## 私たちの基準

ポジティブな環境に貢献する行動の例:
- 他者への共感と優しさを示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを提供し、またそれを潔く受け入れること
- 責任を認め、ミスによって影響を受けた人々に謝罪し、その経験から学ぶこと
- 個人としてだけでなく、コミュニティ全体にとって何が最善であるかに焦点を当てること

許容されない行動の例:
- 性的な言葉や画像の使用、およびあらゆる種類の性的な注目やアプローチ
- 荒らし行為、侮辱的または軽蔑的なコメント、および個人的または政治的な攻撃
- 公的または私的なハラスメント
- 明示的な許可なしに他人の個人情報を公開すること
- プロフェッショナルな環境において、合理的に不適切とみなされるその他の行為

## 執行の責任

プロジェクトのメンテナは、許容される行動の基準を明確にし、施行する責任を負い、不適切、脅迫的、攻撃的、または有害であると判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

この行動規範は、すべてのコミュニティスペース（issue、pull request、discussion）内で適用され、また、個人が公式にコミュニティを代表して公共のスペースにいる場合にも適用されます。

## 執行

虐待、ハラスメント、またはその他許容されない行動の事例は、「confidential（極秘）」とマークされたGitHubのissueを介して、またはメンテナのGitHubプロフィールにある連絡先を通じてメンテナに報告することができます。すべての苦情は、迅速かつ公正に審査および調査されます。

## 帰属

この行動規範は、[Contributor Covenant](https://www.contributor-covenant.org) バージョン 2.1 に基づいて適応されています。オリジナルは https://www.contributor-covenant.org/version/2/1/code_of_conduct.html で入手可能です。
